Description de votre projet

Par Victor Bonny le 09/03/2019 à 09:00

Pour notre cours RIA - Génie logiciel, nous devons créer un jeu en utilisant différents langages web.

Nous avons décidé de faire un jeu de type "running".

Un personnage court et fait des points en avançant le plus loin possible. Tout au long du parcours, il y a des obstacles à éviter. Le concept est le même que Temple Run ou Angry Birds. Nous n'avons pas encore régler quelques détails comme le nombre de vie du personnage, si les points augmentent en fonction de la distance parcourue ou si il y a des items à attraper pour avoir des bonus et le titre définitif n'a pas été choisi.

Le code source de l'application est disponible sur le github du projet.
