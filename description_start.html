Description de notre projet

Par Victor Bonny le 09/03/2019 à 09:00

Durant notre 4ème semestre à la HES-SO de Sierre, nous sommes ammenés à réaliser un projet pour le module Génie logiciel - RIA. Le projet consiste à développer un jeu de notre choix en utilisant différent language dont l'HTML, le CSS et le javaScript.

Nous avons alors décidé de faire un jeu de type "running" avec des obstacles.

Le principe est qu'il y a un personnage qui se déplace en courant. Plus il avance loin, plus il récolte des points. Tout au long du parcours, il y a plusieurs obstacles dont des clôtures en bois. Le concept de notre jeu ressemble "Temple Run" ou encore un ancien jeu a succès "Flappy bird".

Nous allons encore décider si nous allons accorder plusieurs vies à notre héro ou une seule. Nous allons également nous pencher sur l'idée de pouvoir lui donner des points supplémentaires à l'aide d'items qu'il devra attraper.

Le code source de l'application est disponible sur le github du projet.
