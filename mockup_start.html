Mockup de votre projet

Par Victor Bonny le 09/03/2019 à 13:00

Pour le site où se situe notre jeu, nous pensons faire quelque chose de sobre qui met en avant notre jeu.

Il y aurait une page avec une image en fond en relation avec le thème d'arrière plan du jeu. Ensuite, le jeu est dans au centre et il n'y aurait qu'une fenêtre avec le contenu qui change entre le menu, le jeu et la fin.

Premièrement, il y a un menu où le joueur peut sélectionner un personnage et lancer le jeu.

On ne sait pas encore si on fera avec un Glass-Pane par-dessus ou seulement des div.

Le code source de l'application est disponible sur le github du projet.
